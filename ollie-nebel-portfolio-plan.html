Website Strategy Document

Ollie Nebel — Portfolio Redesign Plan

A full-site architecture, content strategy, animation plan, and tech stack for an aerospace-themed portfolio — inspired by thibaultguignand.com and des.de.

1 Analysis — Current Site & Inspirations

🔍 Current Site — What Works

  • Clean content structure (About / Projects / Experience / Achievements)
  • Good domain & personal branding
  • Correct content hierarchy
  • Fast, lightweight

⚠️ Current Site — Gaps

  • No visual personality or theme
  • No animations or interactivity
  • Very minimal content (sparse detail)
  • No standout hero moment
  • Multi-page feels fragmented — no narrative

✦ From thibaultguignand.com

  • Full-viewport sections — each "breathes"
  • Cinematic large typography as art
  • Custom cursor with hover effects
  • Project cards with smooth reveal transitions
  • Minimal nav — almost invisible until needed
  • Work shown with full-bleed visual impact

✦ From des.de

  • Bold, confident section headings
  • Strong use of negative space
  • Systematic grid layouts
  • Clear content hierarchy — nothing competes
  • Smooth scroll with section snap
  • High-contrast text on dark/image backgrounds
2 Visual Identity — Aerospace Theme

🎨 Concept: "Deep Space Operations"

  • Deep near-black backgrounds (#05070f, #0b0f1e) — evoke the void of space
  • Electric blue as the primary accent — control panels, readouts, HUD aesthetics
  • Amber / warm orange as secondary — echoes rocket exhaust and alert states
  • Fine white particle systems for starfields, not distracting but present
  • Monospace type for data/technical elements; refined serif or clean sans for headings
  • Grid lines and coordinate overlays as subtle decorative elements
  • Velocity / trajectory lines in section transitions

3 Site Structure — Single Page, Scroll-Driven

📐 Architecture Decision

  • Single page with smooth scroll — preserves narrative, feels immersive like the inspiration sites
  • Fixed nav with section dots (right side)
  • Each section is full or near-full viewport height — dramatic and deliberate
  • Sections snap softly as user scrolls — guides the experience

4 Animation Plan
✦

Hero Text Reveal

Name letters mask in from below, staggered 30ms each. Subtitle fades in 400ms after. Creates a cinematic opening moment.

🌌

Particle Starfield

Canvas-based background, ~150 particles at varying speeds and opacities. Subtle parallax on mouse move. Fades between sections.

↓

Scroll-Triggered Reveals

Every section element triggers when 20% into view. Fade-up with 0.6s ease-out. Stagger between children for rhythm.

🖱

Custom Cursor

Small circular cursor with a trailing dot. Expands on hover over links. Subtle glow on interactive elements. Strong signature detail.

📋

Project Card Hovers

Card lifts 4px with box shadow, background image fades in, border glow activates. Scale 1.02 on transform for depth.

📈

Timeline Draw

SVG line in experience section draws itself as you scroll down, with nodes "popping" into existence at each entry point.

🔢

Counter Animations

Any numerical stats (scholarship year, competition rank, hours, etc.) count up from 0 when scrolled to — satisfying and draws attention.

⚡

Navigation

Nav hidden on hero, fades in with backdrop blur after first scroll. Active section indicator slides smoothly between nav items.

🛸

Page Transitions

If using multi-page: full-screen wipe with a horizontal bar sweep in deep navy — like a launch sequence. Otherwise smooth scroll between sections.

7 Recommended Build Phases

🗺 Phased Approach

Phase 1
Foundation — Content & Structure
Set up Next.js project, Tailwind config with your colour tokens. Write all copy and gather all content first — design around real content, not placeholders. Build static versions of all 6 sections.
Phase 2
Visual Identity — Theme & Typography
Apply the aerospace colour palette, set up the starfield canvas, choose and integrate fonts (Space Grotesk for headings, Inter for body). Get the hero looking dramatic.
Phase 3
Animation Layer
Add Framer Motion scroll triggers, hero text reveal, project card hovers, timeline draw animation, and counter animations. Add custom cursor last (easy win, high impact).
Phase 4
Polish & Performance
Add mobile responsiveness, test animation performance (reduce particle count on mobile), add page meta / SEO, check accessibility (contrast ratios), and deploy to Vercel with your existing domain.
Phase 5 (Optional)
Advanced Features
Three.js wireframe model in the hero, interactive orbit path in the background, WebGL shader effects, or a blog/writing section for research notes.
